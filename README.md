# The Greatest Letter That's Ever Lived

A simple, beautiful, and customizable Star Wars-style intro crawl for a personal letter.

## Description

This project displays a letter in the iconic opening crawl of a Star Wars movie. It features a pure CSS animated starfield background with multiple layers and shooting stars for a dynamic and immersive experience.

## How to View

1.  Clone this repository or download the files.
2.  Open the `index.html` file in your web browser.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript (empty `script.js` file included)

## Customization

You can easily customize the letter by editing the `index.html` file. The text is located within the `<div class="crawl">` section. The background can be further customized by editing the `style.css` file.
